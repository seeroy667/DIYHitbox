# DIYHitbox
A home made Hitbox for playing fighting games.

This repository includes the solidworks files used for the case.

Things needed for the project:
    - 3D printer
    - Leonardo arduino
    - Micro USB cable
    - Screws

The solidworks models are mine based on the CAD files made from a fan as close as they can be to reality: 

CAD files source: https://www.reddit.com/r/fightsticks/comments/ybe9co/cad_hitbox_button_layout_pdf_drilling_template_an/ 
Nintendo library for emulating: https://github.com/lefmarna/NintendoSwitchControlLibrary
